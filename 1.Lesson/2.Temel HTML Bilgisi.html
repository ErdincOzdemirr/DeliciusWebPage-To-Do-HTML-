Komut Yapısı

tagların özelliklerine dikkat. id ile tag a isim verilebilir. style ile komut içi css uygulaması yapılarak renk boyut konum belirlenebilir. tagların açılış ve kapanışını unutma. Komutların okunabilirliğini arttırmak için içiçe girintiye dikkat et. ex:

deneme için tag yapısı

temel Html komutları açıklanmıştır. html tagı sayfa start finish belirtir. head arama motorları içerik. body ziyaretçiye gösterilecek metacharset utf 8 tr karakterler için. hx büyük başlıklar oluşturmamızı sağlar. p paragraf br alt satıra geçmemizi sağlar

Beğendiğin sayfadaki html yapısını incelemek için kaynak kodu görüntüle yapısı ile inceleme yapabilirsiniz.

html sık kullanılan özllikleri ; id : taga özel isim verme class : bir css komutunu birden fazla html komuduna kullanılacaksa. style : komuta css kodu yazılmasını sağlar. title : komut üzerine gidince bilgi kutucuğu çıkar.

ör

Tag içi özellik kullanımı hakkında. yazının üstüne imleç ile gelindiğinde bilgi kutucuğu açılır.
